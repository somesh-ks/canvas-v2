# AGENTS.md

Canvas V2 is a React + Vite single-page presentation/analytics web app styled with Tailwind.

- Package manager: **Bun** (`bun install`, `bun run dev`, `bun run build`, `bun run typecheck`).
- Tests have no runner script; run individually with `node tests/<name>.test.js`. See [docs/TESTING.md](docs/TESTING.md).
- Visual/UX conventions live in [DESIGN.md](DESIGN.md) — consult it before adding or changing UI.
